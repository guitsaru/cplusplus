h1. cplusplus

h2. What

Rubigen generators for C++ projects.

h2. Installing

<pre syntax="ruby">sudo gem install cplusplus</pre>

h2. The basics

<pre>cplusplus project_name</pre> to create a new C++ project

<pre>script/generate class class_name</pre> to create a new class - Includes .cpp file, .h file, and a cppunit unit test.

<pre>script/generate file file_name</pre> to create a new file - Includes .cpp file, .h file, and a cppunit unit test.

h2. How to submit patches

Fork the project on github and send a pull request.

  • github: "github.com/guitsaru/cplusplus/tree/master":http://github.com/guitsaru/cplusplus/tree/master

h3. Build and test instructions

<pre>cd cplusplus rake test rake install_gem</pre>
